# Lecture 5: Magic 8-Ball with Fortune Cookie Sayings

This activity extends a Magic 8-Ball example by adding Fortune Cookie sayings as an alternative to Magic 8-Ball answers. The application randomly selects between the two lists when the user asks a question, creating an interactive and visually appealing experience with a shaking animation.

## Features

* **Dual Answer Arrays**: Contains both Magic 8-Ball answers (affirmative, non-committal, and negative) and Fortune Cookie sayings
* **Random Selection**: The `getAnswer()` function randomly selects between the two lists with a 50/50 probability
* **Interactive Input**: Users enter a question in a text box and click the "Ask" button
* **Shake Animation**: The Magic 8-Ball image shakes for 1.5 seconds when the user clicks "Ask"
* **Console Logging**: All interactions are logged to the browser console for debugging
* **Responsive Design**: Clean, centered layout with styled input and button elements
