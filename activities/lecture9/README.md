# L9 Activity - Form Focus and Blur Events

This activity extends the lecture example by adding `focus` and `blur` event listeners to the First Name, Last Name, and Email inputs. The script uses a loop to apply a consistent interaction pattern and updates the input styling for clearer visual feedback.
