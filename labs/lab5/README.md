# Lab 5: Recipe Recommender App (RecipeFind)

This is an individual CSCI 3172 lab submission. RecipeFind is a web application that recommends recipes based on user-entered ingredients, dietary preferences, and intolerances. It uses a Netlify Function written in Node.js to securely call the Spoonacular API, then renders interactive recipe cards and details on the client side.
